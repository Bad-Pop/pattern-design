Api documentation

Articles

GET /api/v1/article/{label} Récupérer un article.

Attributs:

  Attribut Type Description
RequiredlabelStringNom de l'article à récupérer.

GET /api/v1/article/ Récupérer tous les articles.

Aucun attribut ni json juste get de l'url.

POST /api/v1/article/ Créer un article.

Sample:

application/json


{
	"label" : "ArticleTest1",
	"type" : "LUXE",
	"color" : "ROUGE",
	"size" : "XXL"
}

DELETE /api/v1/article/{label} Supprimer un article.

Attributs:

  Attribut Type Description
RequiredlabelStringNom de l'article a supprimer.

Catalogues

GET /api/v1/catalog/{label} Récupérer un catalogue.

Attributs:

  Attribut Type Description
RequiredlabelStringNom du catalogue à récupérer.

GET /api/v1/catalog/ Récupérer tous les catalogues.

Aucun attribut ni json juste get de l'url.

POST /api/v1/catalog/ Créer un catalogue.

Sample:

application/json


{
	"label" : "CatalogTest1",
	"articles" : null
}

DELETE /api/v1/catalog/{label} Supprimer un catalogue.

Attributs:

  Attribut Type Description
RequiredlabelStringNom du catalogue à supprimer.
